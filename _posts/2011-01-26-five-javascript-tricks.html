Javascript is a great language. I've written a short list of five useful javascript features and techniques that are a must-know.

1. Closures

Prevent setting global variables by putting all your code in a function. Like this: {% highlight javascript %} (function(){ //Everything goes in here //This x variable is not accessible to the outside world. var x="Hello World"; }()); {% endhighlight %}

Usings closures promotes good code structure, protects private variables, and ensures that your code won't interfere with any other scripts running on the page.

2. JSON

JSON (JavaScript Object Notation) is a way to describe objects.

{% highlight javascript %} var me={ "name":"John Smith", "location":[-40.000,150.000], "description":"", "interests":{ "movies":[], "music":["All music"] } }; alert(me.name);//or alert(me["name"]); alert("JSON representation: "+JSON.stringify(me)); alert(JSON.stringify(JSON.parse(JSON.stringify(me)))==JSON.stringify(me)) {% endhighlight %}

3. Prototyping

Have you ever used the new Keyword()? Well when you do, the object returned from the function (provided the function doesn't "return" an object manually), has it's prototype set. {% highlight javascript %} function Message(title,body,imagesrc){ this.title=title; this.body=body; if(imagesrc){ this.imagesrc=imagesrc; } } //If a message doesn't have an imagesrc, it will inherit it from its prototype chain. Message.prototype.imagesrc="default.png"; Message.prototype.alert=function(){ alert(this.title+"\n"+this.body); return this; } var msg=new Message("Title","Body"); msg.alert().alert(); {% endhighlight %}

You can also add methods to objects such as Array, Function, and Object. For example: {% highlight javascript %} Array.prototype.sum=function(){ return this.reduce(function(x,y){return x+y;}) }; alert([1,2,3].sum());//1+2+3=6 {% endhighlight %}

4. jQuery

jQuery is great! I suggest you read more about it in jQuery Fundementals (A great guide), or on the jQuery website.

5. Console

I find using the consoles in Safari, Chrome and Firefox really speed up development. Using console.log() is often easier than messing around with alert().
